# 📘 Campus Management System - Roadmap

##  Final Goal:
Build a complete system that manages:
- Students
- Courses
- Attendance
- Marks
- Reports

---

##  Learning Path:

### Stage 1: Basics
- Input/output
- Variables
- Lists

### Stage 2: Data Handling
- Dictionaries
- Nested data

### Stage 3: Functions
- Modular code
- Reusable logic

### Stage 4: Menu System
- While loops
- User interaction

### Stage 5: File Handling
- Save/load data

### Stage 6: OOP Conversion
- Classes and objects

---

## ⚠️ Rule:
Do not jump stages. Build step by step.
